# Work Order & Parts Picking System

A comprehensive work order management system with parts inventory tracking.

## Features

- ✅ **Work Order Management** - Create, update, archive work orders
- ✅ **Parts Management** - Search and filter parts by number or description
- ✅ **CSV Import** - Import parts from CSV files
- ✅ **Technician Management** - Add, edit, and manage technicians
- ✅ **Status Management** - Customizable status options with color coding
- ✅ **Parts Picking** - Add parts to work orders with quantity tracking
- ✅ **Bulk Updates** - Update statuses for work orders and parts in bulk
- ✅ **Mobile Responsive** - Works on desktop, tablet, and mobile
